# UTF-8 Icon Viewer

A simple and modern web application that allows you to quickly view, search, and copy Unicode (UTF-8) icons.

## Key Features

- **Browse icons by category:** Supports many popular Unicode blocks such as Arrows, Emoticons, Mathematical Operators, Box Drawing, Dingbats, and more.
- **Smart search:** Quickly search for Unicode blocks or Hex codes (e.g., `u+2190`, `1F600`).
- **One-click copy:** Click on any icon to copy it directly to your clipboard.
- **User-friendly interface:** Smooth support for Light and Dark modes.
- **Full character table:** Detailed Unicode code (U+XXXX) display for each character.

## Technologies Used

- **React 19**: A JavaScript library for building user interfaces.
- **Vite**: A fast build tool and development server.
- **CSS3**: Modern UI with animations and theme support.
